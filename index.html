Как играть

  • Общие правила:
  • Выберите от 2 до 6 игроков.
  • Введите имена игроков перед началом игры.
  • Заполняйте очки в строках категорий для каждого игрока.
  • Имена игроков можно менять нажатием на заголовок колонки.
  • Виртуальные кости можно бросить максимум 3 раза за ход.
  • После каждого броска можно выбрать любые кости для переброса: одну, несколько или все 5.
  • Первые 3 броска через виртуальные кости можно записывать только в категории 1–6.
  • Комбинации можно записывать только после 3-го броска.
  • Удвоение x2 работает только для комбинаций и только если кости, участвующие в комбинации, выпали на первом броске и не перебрасывались.
  • Категории 1–6 никогда не удваиваются.
  • П — пара/покер, ПП — две пары, S — сет, MS — малый стрит, BS — большой стрит, F — фулл-хаус, K — каре, M — мусор.
  • Правила Kuzma:
  • Строки 1–6 суммируются и умножаются на 10.
  • Правила Koro:
  • Строки 1–6 считаются без умножения.
  • Если сумма строк 1–6 меньше 0, добавляется штраф -50.
