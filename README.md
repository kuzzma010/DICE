# Покер на костях: Telegram Mini App

Статическое приложение для подсчета очков в игре "Покер на костях".

## Возможности

- выбор от 2 до 6 игроков;
- таблица очков с категориями игры;
- автоматический подсчет строки "ИТОГО / СУММА";
- автоматический подсчет строки "ОБЩИЙ ИТОГ";
- переименование игроков через модальное окно;
- сохранение имен и очков в `localStorage`;
- поддержка Telegram WebApp API: `ready()`, `expand()` и цвета темы;
- работа в Telegram и в обычном браузере.

## Локальный запуск

Откройте `index.html` в браузере.

Также можно запустить простой локальный сервер из папки проекта:

```bash
python -m http.server 8000
```

После этого откройте:

```text
http://localhost:8000
```
